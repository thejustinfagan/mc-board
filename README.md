# 🎯 Mission Control

**Barry & Justin collaboration board.**

Use this like a running command center:
- Create an issue = give Barry a task
- Comment = Barry picks it up on heartbeat
- Close = done/cancelled

---

## ELI5 Project Index (what each project is + why + when)

### 🚛 Fleet Intel — **“Carrier Cheat Codes”**
- **What:** Tells you who a trucking carrier is (size, safety, locations, patterns) so you can sell smarter.
- **Why:** Day job + direct revenue path.
- **When:** Prepping for calls, building target lists, deciding who’s worth chasing.

### 🏪 Reseller Intel — **“Dealer + Service Center Map”**
- **What:** Database of truck dealers/service centers/distributors (who/where/brands/services).
- **Why:** Prospecting + territory strategy.
- **When:** Need leads in a region or want the competitive landscape.

### 🧲 Truck Reseller DB (pipeline) — **“Louisville Lead Vacuum”**
- **What:** The scraper that *builds* the reseller database (finds businesses + pulls contacts/services/brands).
- **Why:** The dataset doesn’t exist cleanly off-the-shelf.
- **When:** Runs in background; you consume the CSV outputs.

### 🆔 VIN Intelligence — **“VIN Upload + Decode”**
- **What:** Import VIN lists and turn them into useful vehicle info.
- **Why:** VIN enrichment is leverage for parts fitment + targeting.
- **When:** You have VINs and want them turned into something usable.

### 🎯 Territory Command Center — **“Territory War Room”**
- **What:** Planning dashboard for territory/market coverage decisions.
- **Why:** Turns “where do we focus” into a visual decision.
- **When:** Quarterly planning, account assignment, expansion decisions.

### ⚡ DC Land Intel — **“Grid + Land Opportunity Finder”**
- **What:** Public datasets about grid/substations/land values for site selection/research.
- **Why:** Experiment: package public infra data into something sellable.
- **When:** Exploring energy/land/site-selection angles.

### 🧭 Mission Control — **“One Place to Tell Barry What to Do”**
- **What:** This board + any dashboards we build.
- **Why:** You shouldn’t have to remember chat context.
- **When:** Any time you want to assign, reprioritize, or check status fast.

### 📡 SentinelClaw — **“Always-On Radar”**
- **What:** Background monitor that collects signals and keeps systems healthy.
- **Why:** Proactive intel instead of manual checking.
- **When:** Mostly invisible unless something important pops.

### 🎮 Battle Dinghy — **“The Game”**
- **What:** The game project.
- **Why:** Long-term product build.
- **When:** When we’re in game-feature shipping mode.

### 🧪 X Simulator — **“Live Rule-Change Demo”**
- **What:** Demo where you say “change the rules” and the system adapts in real time.
- **Why:** Proves real-time collaboration + agent execution.
- **When:** Showing someone what fast AI-powered iteration looks like.
