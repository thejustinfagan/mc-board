# 🎯 Mission Control

**Barry & Justin collaboration board.**

## How to use
- **Pink labels (needs-justin)** = blocked on you — comment or close when handled
- **Green labels (barry-working)** = I'm on it
- **Orange labels (decision)** = needs your call
- **Red labels (blocked)** = stuck on external factor
- **Blue labels (idea)** = pipeline / future work

## How Barry uses this
- Checks issues every heartbeat (30 min)
- Picks up new tasks you create
- Comments with progress/results
- Closes issues when shipped
- Creates new issues for things that need your input

## How Justin uses this
- Open issues list → see what needs attention
- Comment on any issue → Barry picks it up
- Create new issue → it's a task for Barry
- Close an issue → Barry knows it's handled/cancelled
